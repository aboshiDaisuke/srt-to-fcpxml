# SRT Subtitle Editor → FCPXML

Final Cut Proに日本語字幕を読み込むためのブラウザツールです。

## なぜ必要？

Final Cut Proは日本語の自動字幕に非対応です。  
Whisper、VREW、YouTubeの自動字幕などで生成したSRTファイルを、FCPのタイムラインに読み込むために変換が必要です。

## 機能

### 字幕編集
- テキスト・タイムコードの直接編集
- 字幕の分割（カーソル位置で分割）・結合・削除・追加
- 句読点の自動削除（。、！？）

### 動画プレビュー
- 動画とSRTを同時にドラッグ&ドロップで読み込み
- 再生中に該当する字幕をハイライト表示
- 字幕クリックで動画の該当位置にジャンプ

### FCPXML書き出し
- フレームレート選択（23.976 / 24 / 25 / 29.97 / 30 / 59.94 / 60 fps）
- 解像度選択（FHD / 4K / HD / 縦型対応）
- フレームオフセット調整（FCP上のズレをフレーム単位で補正）
- 字幕の縦位置指定

### その他
- 自動保存（ブラウザを閉じても復元可能）
- プロジェクトファイル保存/読み込み
- 文字エンコーディング対応（UTF-8 / Shift-JIS / EUC-JP）

## 使い方

1. ブラウザでエディタを開く
2. 動画ファイルとSRTファイルをドラッグ&ドロップ
3. 字幕を確認・編集
4. 「FCPXML書き出し」→ Final Cut Proに読み込み

## 特徴

- **完全無料** — サブスク・課金なし
- **ブラウザ完結** — インストール不要
- **アップロードなし** — ファイルはブラウザ内でのみ処理
- **オフライン対応** — ネット接続なしでも動作
- **OS問わず** — Chrome / Safari / Firefox / Edge 対応

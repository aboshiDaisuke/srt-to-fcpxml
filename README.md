# SRT Subtitle Editor → FCPXML

Final Cut Proに日本語字幕を読み込むためのブラウザツールです。

## なぜ必要？

Final Cut Proは日本語の自動字幕に非対応です。  
Whisper、VREW、YouTubeの自動字幕などで生成したSRTファイルを、FCPのタイムラインに読み込むために変換が必要です。

## 機能

### 字幕編集
- テキスト・タイムコードの直接編集
- 字幕の分割（カーソル位置で分割）・結合・削除・追加
- **複数行（複数段）の字幕入力**（Shift+Enterで改行）
- 句読点の自動削除（。、！？）
- **文字数オーバー警告**（各行20文字超えでハイライト）
- **検索と一括置換**

### 動画プレビュー・波形表示（🆕）
- 動画とSRTを同時にドラッグ&ドロップで読み込み
- **独自の音声波形表示機能（Web Audio API）**
  - スペースキーなどで動画再生と完全にシンクロして波形がスクロール
  - クリックで指定した時間にシーク
  - `↔＋/ー` で波形の時間軸ズーム拡大縮小
  - `↕＋/ー` で波形の高さを強調
- 波形上の**字幕ブロックの端（IN点/OUT点）を直接ドラッグ**して直感的なタイミング調整が可能
- 再生中に該当する字幕をハイライト表示
- **オートスクロールのON/OFF切り替え**

### 書き出し・保存
- **FCPXML書き出し**
  - フレームレート選択（23.976 / 24 / 25 / 29.97 / 30 / 59.94 / 60 fps）
  - 解像度選択（FHD / 4K / HD / 縦型対応）
  - フレームオフセット調整（FCP上のズレをフレーム単位で補正）
  - 字幕の縦位置指定
- **SRTファイルの直接保存**（他の編集ソフトへの連携用）

### その他
- 自動保存（ブラウザを閉じても復元可能）
- 文字エンコーディング対応（UTF-8 / Shift-JIS / EUC-JP）

## 使い方

1. ブラウザでエディタを開く
2. 動画ファイルとSRTファイルをドラッグ&ドロップ
3. 字幕を確認・編集（検索置換、分割結合、文字数チェックなどを活用）
4. 用途に合わせて書き出し
   - FCPで使う場合: 「FCPXML書き出し」→ Final Cut Proに読み込み
   - 他のツールで使う場合: 「SRT保存」で標準のSRTとして保存

## 特徴

- **完全無料** — サブスク・課金なし
- **ブラウザ完結** — インストール不要
- **アップロードなし** — ファイルはブラウザ内でのみ処理
- **オフライン対応** — ネット接続なしでも動作
- **OS問わず** — Chrome / Safari / Firefox / Edge 対応
